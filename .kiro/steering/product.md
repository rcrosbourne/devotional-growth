# Product

A Laravel starter kit for building authenticated web applications. It provides a complete user authentication system including registration, login, password reset, email verification, two-factor authentication (TOTP), and user profile management. The frontend is a React SPA served via Inertia.js with a sidebar-based dashboard layout.

This is a starter kit / boilerplate — not a finished product. It's meant to be extended with domain-specific features.
